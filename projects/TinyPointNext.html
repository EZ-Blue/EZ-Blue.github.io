TinyPointNeXt: Reducing Model Parameters for Efficient Dynamic Object Segmentation

Introduction

In modern robotics and autonomous systems, real-time detection of moving objects, particularly humans, is critical for safe navigation and interaction. Traditional approaches require computationally expensive models that strain resource-constrained platforms like mobile robots and drones. Our project addresses this challenge by developing TinyPointNeXt, an optimized variant of the PointNeXt architecture that achieves state-of-the-art accuracy while dramatically reducing computational requirements for human detection in point clouds.

Objective: To develop an efficient point cloud processing model that can accurately detect humans in dynamic environments while being lightweight enough for deployment on resource-constrained devices, enabling real-time processing (<0.1 seconds inference) essential for safety-critical applications.

Methods

Dataset and Architecture

We conducted experiments on two datasets:

  • HeliMOS Dataset: 12,188 labeled point clouds from KAIST Campus for dynamic object segmentation
  • Custom Human Detection Dataset: 7,011 point cloud clusters (4,614 human, 2,397 false positive) collected at Carnegie Mellon University

Our baseline model was PointNeXt-S, which we systematically modified to create five TinyPointNeXt variants:

  1. Architecture 1-2: Reduced MLP width from 32 to 24 and 16 units
  2. Architecture 3-4: Reduced Set Abstraction blocks from 6 to 5 and 4 blocks
  3. Architecture 5: Reduced MLP layers within each block from 2 to 1

Training and Evaluation

We employed comprehensive ablation studies to evaluate:

  • Data augmentation techniques (geometric, noise, sampling, and combined)
  • Architectural modifications across 48 different configurations
  • Performance metrics: accuracy, model parameters, and inference time

Discussion

Social Impact and Applications

TinyPointNeXt addresses critical challenges in robotics and autonomous systems:

  • Enhanced Safety: Enables real-time human detection for collision avoidance in dynamic environments
  • Accessibility: Makes advanced perception systems feasible for low-cost robotics platforms
  • Energy Efficiency: Reduced model size extends battery life for mobile applications
  • Scalable Deployment: Facilitates widespread adoption in resource-constrained scenarios

Technical Contributions

Our work demonstrates that architectural efficiency improvements can outperform increasing model complexity:

  • Smaller models exhibit better generalization by reducing overfitting
  • Task-specific optimization yields superior performance for specialized applications
  • Strategic parameter reduction acts as implicit regularization

Conclusion

TinyPointNeXt successfully achieves the project objective by delivering an efficient, accurate model for human detection in point clouds. Our 4-Block Encoder variant uses only 19% of the original model's parameters while achieving 93.86% accuracy and 2.47ms inference time, making it suitable for real-time applications on resource-constrained platforms.

The success of our approach challenges conventional assumptions about model complexity, proving that thoughtfully designed smaller models can outperform larger ones for specialized tasks while enabling practical deployment in safety-critical applications.
